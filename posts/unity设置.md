#### 设置shadow distance
Edit -> Project Settings -> Graphics 点击 URP-HighFidelity，会出现文件，点击文件，Ispector 面板中，设置 Shadow / Max Distance 

#### 解决图片锯齿
勾选图片文件的 Generate Mip Maps 选项
